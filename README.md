# Lore Playground Gallery - Simple Version

A minimalist image gallery displaying AI-generated artwork.

## Features

- Dense grid of thumbnails from GitHub
- Click to view image details in popup
- Copy seed, prompt, and negative prompt
- Clean design with metallic "LORE" logo

## Deployment

1. Create a Netlify account at [netlify.com](https://netlify.com)
2. Drag and drop the `Lore-Gallery-Simple` folder onto Netlify
3. Your site will be deployed instantly

## Image Source

Images are loaded from the GitHub repository `T0nt0o/Promptofolio_Lore`:
- thumbnails1, thumbnails2, thumbnails3, thumbnails4 branches

## Notes

- Data is embedded directly in the HTML (no external CSV file needed)
- Simple and reliable - no API calls, no complex JavaScript
- Just thumbnails (no big images)
